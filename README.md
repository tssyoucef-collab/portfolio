## ✨ Features (50+)

- **Security**: CSP with Nonces, Fail2Ban IP blocking, Idempotency keys, Audit logging, XSS sanitization, CSRF protection.

## 🛠️ Tech Stack

- **Backend**: Node.js, Express, Firebase Admin SDK.
- **Frontend**: Vanilla JS, GSAP, Firebase Auth (Google/GitHub/Facebook/Email).
- **Database**: Firestore.

## 📦 Installation

1. Clone the repo.
2. `npm install`
3. Create `.env` file (see `.env.example`).
4. Place your Firebase service account key at the path specified in `.env`.
5. `npm start`

## 🔐 Security Notes

- Never commit your service account key.
- Use Render Secret Files or environment variables for secrets.
- Rotate your Firebase private key if it has been exposed.
